Приложение по учету расходов.
Пользователь может создавать категории расходов, заносить свои траты. Показывается суммарное значение трат.
Данные кторые обрабатываются приложением
id (Целое число) — уникальный идентификатор записи (Primary Key).
amount (Вещественное число) — сумма расхода.
category (Строка) — категория трат (например: "Продукты", "Транспорт", "ЖКХ", "Развлечения").
date (Дата/Время) — дата совершения покупки.
description (Строка) — краткий комментарий или пояснение к трате (необязательное поле).
Действия в приложении
Создание (Create): Добавление новой записи о расходе через HTML-форму (ввод суммы, выбора категории и даты).
Чтение (Read): Просмотр таблицы/списка всех своих расходов. Отображение общей суммы всех трат.
Обновление (Update): Редактирование существующей записи (например, если пользователь ошибся при вводе суммы или категории).
Удаление (Delete): Удаление ошибочно созданной записи о расходе.
